# Portfolio Project

This is a simple web portfolio project. The repository contains:

- `index.html` - the main HTML file for the portfolio website.
- `package.json` - project metadata and dependencies (if any).
- `server.js` - a Node.js server script for serving the site.
- `images/` - directory containing portfolio images.

## Getting Started

1. Clone the repository.
2. Run `npm install` if dependencies are needed.
3. Start the server with `node server.js` or open `index.html` directly in a browser.
